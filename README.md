# Pink Pong Mobile 📱

This repository demonstrates the transition from desktop-first logic to **Mobile-First UX**. It is a functional application of mobile-specific web APIs, ensuring high-performance gameplay on touch-screen devices.

### 🛠️ Real-World Application & Skills
- **Touch Event Integration:** Replaced traditional keyboard listeners with `touchstart` and `touchmove` events to handle low-latency paddle control on mobile browsers.
- **Viewport Optimization:** Implemented dynamic scaling to ensure the HTML5 Canvas maintains its aspect ratio across various screen resolutions (iPhone, Android, Tablets).
- **User Experience (UX):** Adjusted game physics and paddle sensitivity to account for finger-tracking rather than key-presses, providing a natural feel for handheld users.
- **Cross-Browser Compatibility:** Tested for consistent performance across mobile Safari and Chrome.

### 🚀 Tech Stack
- **Language:** JavaScript (ES6+)
- **Mobile API:** Touch Events API
- **Rendering:** HTML5 Canvas
- **Styling:** CSS3 Media Queries
